Book Ease Salon Booking System

Final Year Project

A full-stack salon booking and administration system developed as my final year university project. The application combines a responsive Vue.js frontend with a Spring Boot REST API, allowing customers to book appointments while helping salon administrators manage services, staff schedules and bookings.

  • Created a customer booking journey for browsing salon services, selecting staff members and choosing available appointment times.
  • Developed an administration dashboard for managing salon information, services, staff schedules and customer bookings.
  • Built a booking engine that checks staff working hours, availability rules and time off before confirming an appointment.
  • Added validation to prevent conflicting and overlapping bookings.
  • Implemented role-based access for customers, staff members and administrators.
  • Connected the Vue.js frontend to the Spring Boot backend through REST API requests.
  • Used Pinia to manage application state across customer and administration booking workflows.
  • Designed a responsive interface suitable for desktop and mobile devices.

Java Board Game – Software Design Project

University Project

A configurable console-based board game developed in Java to demonstrate object-oriented software design and the practical use of design patterns. Players move through different states while competing to reach the end of the board.

  • Developed configurable gameplay with small and large boards, two or four players, and single or double dice options.
  • Implemented player states for home, board, tail and completed positions.
  • Added collision rules that allow players to send opponents back to their home position.
  • Applied the Factory pattern to create different board, player and dice configurations.
  • Used the State pattern to manage player behaviour at different stages of the game.
  • Used Strategy and Observer patterns to separate collision behaviour and console output from the main game logic.
  • Added a Facade to simplify game setup and execution.
  • Created predefined scenarios with fixed dice sequences to demonstrate and test different game rules consistently.
  • Tracked individual player moves, total moves and the winning player.
Java Object-Oriented Programming Design Patterns Factory Pattern State Pattern Strategy Pattern Observer Pattern Facade Pattern

E-commerce Backend – Software Architecture Project

University Project
